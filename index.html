Caprice libraries provide a plethora of functionality in regards to what the C & C++ standard defines, each library is designed to minimize dependencies on operating-system specific interfaces and instead depend on a strict way to perform system calls. The caprice libraries are an excellent choice for developers looking for standard libraries on embedded systems, or for OS developers.
